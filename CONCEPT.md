# Farming Evolution — Modpack Concept

## Elevator pitch
Minecraft modpack про реальную историю земледелия: от диких несъедобных растений
до генной инженерии. Прогрессия — через селекцию, а не через шахты и заводы.
Технологии существуют только чтобы ускорить работу с растениями.

Вдохновение: Regrowth (1.7.10) — аналогов для современных версий нет.

## Core mechanic
- Старт: мир без съедобных ванильных источников еды
- Единственный путь к еде — селекция диких растений
- Прогрессия: ручной отбор → скрещивание → генетика (AgriCraft)
- Технологии разблокируются через прогресс с растениями, не наоборот

## Progression eras (draft)
1. **Wild** — только дикие растения, ручной отбор, нет ни одной ванильной еды
2. **Cultivation** — базовое земледелие, улучшение почвы, простые инструменты
3. **Breeding** — скрещивание сортов, ирригация, первые механизмы
4. **Genetics** — прямое управление геномом, автоматизация

## Locked decisions
- **Version**: 1.20.1 (Forge, максимальная мод-поддержка)
- **Core mod**: AgriCraft
- **Scripting**: KubeJS
- **Distribution**: CurseForge + Modrinth

## Open questions (→ GitHub Discussions)
- **#1 Driving force**: Ages (SevTech-style gates) vs Village survival
  (Minecolonies, не дать жителям умереть от голода)
  → рассматриваем две параллельные версии
- **#2 Vanilla food lockout**: как именно блокируем ванильную еду?
  (KubeJS скрипт / кастомная генерация мира / оба)
- **#3 Animal breeding**: параллельная ветка прогрессии через животных?
  Аналог растительной селекции но для скота.
- **#4 Hostile mobs**: убрать стандартных (creepers/skeletons) / оставить
  только опасных диких зверей? Позиция: убрать — они просто раздражают,
  не вписываются в концепцию.

## Out of scope (v1)
- Кастомные текстуры
- Мультиплеер-специфичный контент
- Мобы/dungeons как основной контент
