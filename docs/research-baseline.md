## AgriCraft 1.20.1 — что умеет, что не умеет

**Версия:** 4.0.6 Forge (ноябрь 2025), стабильная, активно поддерживается.

### Геномная система
- 7 характеристик: Growth, Gain, Strength, Resistance, Fertility, Mutativity, Species
- Менделевская генетика: доминантный аллель определяет результат
- Диапазон 1–10 (настраивается в конфиге)
- Семена с разными генами не стакуются — боль в инвентаре

### Скрещивание
- Нужны Cross-Crops (× на грядке) + 2 соседних взрослых растения
- Результат: копия родителя / улучшение характеристик / новый вид (через mutation.json)
- Мутации определяются JSON-файлами: `parent1 + parent2 → child` с вероятностью
- При плохом освещении — молча не работает (известный баг)

### Настройка
- Всё через JSON: требования к почве, освещению, сезонам, дропы, стадии роста
- Нативная интеграция с Serene Seasons через поле `seasons` в plant.json
- Команды для тестирования: `/agricraft_seed all`, `/agricraft_seed distinct`

### Жёсткие ограничения (важно для дизайна)
| Проблема | Влияние |
|---|---|
| **Автоматическая уборка сломана** | Industrial Foregoing и Mystical Agriculture харвестеры не работают с AgriCraft на 1.20.1. Автоматизация поздней игры под вопросом |
| **Farmer's Delight × AgriCraft сломан** | FD-культуры не входят в систему генетики из коробки. Нужны кастомные plant.json |
| **Нет биом-специфичных мутаций** | Нельзя написать "это скрещивание работает только в джунглях". Можно обойти через требования к почве |
| **Конфиг не синхронизируется на сервер** | При мультиплеере — ручная раздача |

## Уроки из Regrowth

### Почему зашло
- Новая механика: руды добываются через культуры, не через шахты
- Эмоциональная дуга: пустошь → восстановление мира
- Quest book объяснял каждый шаг — игроки не лезли на вики
- Рецепты через MineTweaker (аналог KubeJS) форсировали использование ВСЕХ модов
- Несколько валидных путей прогрессии (магия vs технологии)

### Что было плохо (не повторять)
- **AFK-гринд у грядок** — культуры росли слишком медленно в ранней игре
- **Визуальная монотонность** — 30–60 часов в пустыне без смены картинки
- **Bottleneck на Mystical Agriculture** — руды только через resource seeds, всё зависело от одной системы
- **Резкие спайки сложности** на переходах между магическими системами

## KubeJS food lockout — реализуемо ли?

**Да, полностью.** KubeJS + LootJS на 1.20.1 покрывают:

- Убрать дроп пшеницы из травы
- Убрать яблоки с дубов
- Убрать дроп мяса с мобов
- Убрать все рецепты с едой
- Сделать предметы несъедобными (убрать food properties)

**Что нельзя сделать через KubeJS** (нужен datapack):
- Убрать руды/структуры из генерации мира
- Изменить биомы и фичи

Вывод: food lockout с первых минут — технически решаемая задача.
